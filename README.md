#### ⚙️ Executando o Front End local
<br/>
• Clone esse repositório <br/> 
• <code>npm intall</code> para instalar todas as dependências requeridas<br/>
• <code>npm run dev</code> para rodar o projeto no servidor local<br/><br/>
